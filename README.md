# 공주민제(DCA) 문서저장소

이 저장소는 공주민제(DCA: Distributed Citizen Assets) 사상·정책 문서의
**정본(코어) + 정책모듈 + 확산키트 + 비판/반비판 아카이브 + PDF 자료실**을 한 곳에 보관한다.

## 빠른 이동
- 시작: /start/
- 정본(코어): /core/
- 정책모듈: /policy/
- 거버넌스: /governance/
- 비판/반비판(레드팀): /debates/
- 조직별 확산키트: /kits/
- PDF 자료실: /pdfs/
- 참고문헌/리서치: /research/

## 운영 원칙(요약)
1) 정본(core)은 안정적으로 유지한다. 큰 변경은 /wip/에서 검토 후 반영한다.
2) 비판/반비판(debates)은 정본과 분리한다(정본성/채택 가능성 보호).
3) PDF는 /pdfs/ 하위에 분류 보관하고 /pdfs/index.md에 링크를 추가한다.
